

Fragrance Finder.

1. User can see the list of shops for that product and their prices
2. User can see similar perfumes (in terms of ingredients)
3. User can see new perfumes coming to the market
4. User can see combinations of perfumes

5. User can see all the perfumes from that brand
6. User can see all the perfumes that have the same top notes, middle notes and base notes (fragrantica)
7. User can see all the perfumes that have the same main accords.
